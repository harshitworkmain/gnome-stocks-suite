# Global Market Data APIs

- **Finnhub (finnhub.io):**  A popular free stock API with real-time US quotes and global coverage (60+ exchanges).  Finnhub offers a **generous free tier** (60 calls/minute) and even free WebSocket streaming (up to 50 symbols)【15†L67-L72】.  It provides live quotes, fundamentals, crypto/forex, and more.  US stock data is genuinely real-time, while many non‑US exchanges are available (often 15‑min delayed)【15†L67-L72】【15†L101-L110】.  Finnhub’s low latency (40–60ms) and streaming support make it ideal for frequent updates【15†L67-L72】【50†L297-L305】.  
- **Yahoo Finance (unofficial):**  Although not an “official” API, Yahoo Finance endpoints (e.g. the query1.finance.yahoo.com REST API) remain widely used to fetch stock quotes for **global symbols**.  No API key is needed, but these calls are undocumented and can change without notice.  Many developers use libraries like `yfinance` or direct JSON endpoints to get quotes.  This is *free*, but beware: it’s not guaranteed by Yahoo (subject to rate-limits and blocking).  Typical usage yields delayed quotes (e.g. “Nasdaq Real Time Price” for US tickers)【17†L113-L117】.  
- **Alpha Vantage (alphavantage.co):**  Provides stock, FX, and crypto data with a free API key.  *Free tier:* only **25 API requests per day**【4†L55-L60】, with ~5 requests/minute.  Real-time US data is limited (15-min delayed on free tier). This is **not very suitable for live widgets** (too few calls, see support FAQ【4†L55-L60】).  
- **Twelve Data (twelvedata.com):**  Covers stocks (US and global), forex, crypto. The *Basic (free) plan* allows **800 API calls per day** (about 8 calls/sec)【35†L223-L232】.  Free tier includes real-time US stock quotes, forex and crypto, as well as 20+ market exchanges.  For example, one can fetch minute-level or quote endpoints via REST or WebSocket (the free WS tier provides 8 trial connections)【35†L223-L232】. Note: real-time coverage on free tier is primarily US stocks/ETFs.  
- **Mboum (mboum.com):**  A stock/options data API (from the makers of a stock screener) that provides US/international quotes.  Mboum offers a **free API key** (with limited daily calls) and global equity coverage (including US, indices, funds, crypto, etc.)【43†L16-L24】【45†L107-L116】.  Its endpoints return real-time quotes, historical prices, analyst ratings, and more (see [Mboum docs](https://docs.mboum.com/)).  Many developers cite a free tier of ~20 requests/day【48†L101-L104】, but exact limits should be confirmed on signup. Mboum’s rate limit is 15 requests/sec (very high)【45†L122-L130】.  
- **Financial Modeling Prep (FMP – financialmodelingprep.com):**  Offers real-time and historical data.  The *free tier* covers basic end-of-day data (250 calls/day)【39†L126-L135】.  Paid plans (e.g. $19/mo) unlock unlimited real-time quotes and WebSockets【50†L419-L428】. Latency is low (≈35–50ms)【50†L265-L274】, but note the free plan’s 250/day limit and lack of true real-time on free.  
- **Marketstack (marketstack.com):**  A REST API for global stock prices. *Free plan:* **100 requests per month**【41†L64-L72】 (essentially only end-of-day US data). Real-time intraday is only on paid plans. Thus, the free tier is very limited (100/mo) and delays (~15+ minutes), making it impractical for live quotes beyond testing.  
- **Others:**  Several other APIs have free tiers worth exploring (see e.g. the [Mboum API blog](https://mboum.com/blogs/10-best-stock-market-apis-2026)).  For example, **EOD Historical Data** (eodhd.com) offers a free 20/day plan and US stock/international data; **SteadyAPI** and **RapidAPI aggregators** also exist. Alpaca Markets provides free US trading data (US-only). In general, for **truly real-time** updates, Finnhub’s WebSocket or trading-platform APIs (see below) are best among free offers.

# Indian Market Data APIs

- **Dhan (DhanHQ platform):**  Dhan offers a developer platform (DhanHQ) with free trading APIs and real-time market data for Indian exchanges【53†L283-L292】.  DhanHQ claims “real-time market data” for NSE/BSE and free algo trading APIs【53†L283-L292】.  (Note: while trading APIs are free, certain data endpoints require a data subscription ~₹499/month【52†L1-L7】.)  You must register and generate an API token via your Dhan account. Dhan uses low-latency infrastructure, designed for automated trading.  
- **Groww API:**  Groww provides trading APIs (for NSE/BSE) to registered Groww users.  For example, the “Get Quote” endpoint (`/v1/live-data/quote`) returns live market snapshots (last price, OHLC, volume, etc.) for NSE/BSE symbols【22†L39-L47】.  These endpoints require an OAuth token (from Groww login) and are free for account holders.  Groww supports Indian stocks, indices (e.g. NIFTY, SENSEX), F&O, and commodities.  With Groww’s API you can fetch up to 50 instruments in one request (batch LTP)【22†L185-L193】.  
- **ICICI Direct – Breeze API:**  ICICI Direct’s Breeze API offers free access to NSE/BSE data. It includes streaming live data and 3 years of intraday history. The key feature is: “**Free Of Charge** – No charges at all to connect your systems, create apps, or access historical data.”【28†L269-L276】.  You can obtain an API key by opening an ICICI account. Breeze provides WebSocket streaming of LTP/OHLC for NSE/BSE (and derivatives) with sub-second updates.  
- **Fyers API:**  Fyers (another Indian broker) provides free API access for account holders.  Fyers advertises **“free”** trading and market data APIs, supporting Indian equities and F&O. For instance, Fyers states up to **1,00,000 requests per day** and order placement <75ms【37†L143-L151】.  You can generate an API key via a free account. The Fyers API offers REST endpoints and WebSockets for real-time tick data on NSE/BSE symbols.  
- **NSE/BSE Web APIs:**  There is no official free public API from the exchanges. Some developers resort to scraping NSE/BSE websites or using third-party proxies (e.g. Yahoo Finance’s `.NS` suffix for NSE). For example, Yahoo symbols like `RELIANCE.NS` or `^NSEI` can fetch Indian quotes via unofficial Yahoo endpoints. But for robust apps, using broker-provided APIs (above) is more reliable.

# Summary

For a GNOME 42.9 extension on Ubuntu 22.04, **Finnhub** is an excellent free global source (real-time US, websockets, 60 req/min)【15†L67-L72】.  For **Indian markets**, broker APIs like **DhanHQ, Groww, ICICI Breeze, Fyers** offer real-time NSE/BSE data at no extra cost (with account registration)【22†L39-L47】【28†L269-L276】【37†L143-L151】【53†L283-L292】.  Note that many of these require obtaining an API key/token from the provider.  Yahoo Finance’s unofficial API can be a lightweight fallback for global quotes.  In choosing APIs, balance latency (websockets help), coverage (some cover 60+ global exchanges) and usage limits (free tiers often cap requests).  Among purely free options, Finnhub (global) and the Indian broker APIs (for NSE/BSE) give the best combination of real-time data and generous access.
